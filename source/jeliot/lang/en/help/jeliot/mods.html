Jeliot Activities

This Module add Jeliot 3 animations activities to your Moodle course.Every activity consists of:

Name
Name of the activity as the student will see it in the main course page
Description
Introduces the activity to the students.
File
The source code that will be animated. The module requires the file to be added to the file repository of your moodle course. To upload and/or choose a file click in the button.
Display Source Code
If selected, the activity will display the source code when the student views the activity
Metadata
List here the concepts that are new for the students in this activity. This feature is not in use yet, but it can be used later.
Ask questions
Tell Jeliot 3 to ask questions during the animation
Points awarded
Amount of points awarded by the student by watching the whole animation and correctly answering the questions in the animation (NOT Working yet).
Mode of adaptation
This feature is selected when the adaptive version of Jeliot is used
