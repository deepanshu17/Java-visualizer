Troubleshooting Java Web Start

If clicking the "Start Jeliot" link does not work, it can be one or more the following reasons. Please check that you have the right setup in your computer

Browser MIME types
For the browser to start jeliot it has to know in advance that JNLP files are to be opened by Java Web Start program (javaws). To do so you have to proceed as follows:
  1. Go to the preferences or options of your favorite browser and select the MIME types configuration (e.g., in Firefox Preferences->Downloads->Download Actions->View and Edit Actions).
  2. Check that there is a MIME type with the name jnlp, application/x-java-jnlp-file and that it opens with the java webstart executable (i.e.,javaws).
  3. Add it if it is not there!
Java Installed
Please, be sure that Sun Java is installed in your system.

If none of these solutions work, please read the Unofficial FAQ, or write to Jeliot Team.
